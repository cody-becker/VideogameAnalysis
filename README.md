# Video Game Data Analysis

This project analyzes the Backloggd Games dataset from Kaggle, exploring platform counts, ratings, release trends, and wishlist behavior.

## Dataset
Source: https://www.kaggle.com/datasets/matheusfonsecachaves/popular-video-games

The dataset includes:
- Game titles
- Release dates
- Developers
- Platforms
- Genres
- Ratings
- Wishlist counts

## What This Notebook Covers
- Data loading and inspection
- Feature engineering:
  - Platform count
  - Wishlist numeric conversion
  - Release year extraction
- Exploratory analysis:
  - Platform count distribution
  - Average rating by platform count
  - Release year trends
  - Wishlist trends (2015+)

## How to Run
1. Clone the repo  
2. Install dependencies  
3. Open the notebook in VS Code or Jupyter  
4. Run all cells
